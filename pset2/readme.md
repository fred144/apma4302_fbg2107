# HW2

Makefile runs all the code. 

the folder `writeups/` contains writeups for each question, latex source and pdf
- the main write up is called `main.pdf` and the source is in `main.tex`.
- `bvp_hdf5_files/` contains the hdf5 files produced by the convergence runs for Q3 as well as the Q4 runs with specific flags
- and `q4_stdouts/` contains the standard output of the Q4 runs, which include the convergence history and reason for convergence/divergence.
